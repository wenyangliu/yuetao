## 粤淘网
    这是一个关于租房的App项目，项目有以下几个模块
        1.登录注册
        2.发布
        3.首页
        4.我
        5.信息
        6.预约

## 如何访问

* 下载
    ```
       git clone https://github.com/wenyangliu/yuetao.git 
    ```
   
* 进入目录
    ```
        cd yuetao
    ```

* 安装依赖
    ```
        npm install
    ```
    
* 启动项目
    ```
        node server.js
    ```
    
* 访问
    ```
        http://127.0.0.1:16926/文件夹/文件夹名
    ```
